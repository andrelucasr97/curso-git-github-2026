git init. = cria um rep git na pasta atual
git status = mostra o status do trabalho (de cada arquivo)
git add nome_arquivo = add arquivo para ser commit
git commit -m = faz commit do que esta sendo feito
git log = mostra todo histórico de commit

#FLUXO DE TRABALHO:
1.criar diretório do projeto
2.iniciar o git dentro do diretório criado (git init.)
3.editar ou criar arquivos
4.git status
5.git add "nome arquivo"
6.git status
7.git commit -m "mensagem qualquer"
8.git status
9.volta para o passo 3
